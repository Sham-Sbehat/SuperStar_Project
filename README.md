# SuperStar - إدخال الطلبات

مشروع React + Vite + TypeScript بأحدث الإصدارات، مُحسّن للأداء، مخصّص لإدخال الطلبات.

## المتطلبات

- Node.js 20+
- npm أو pnpm أو yarn

## الإعداد

```bash
npm install
```

## التشغيل

```bash
# وضع التطوير
npm run dev

# بناء للإنتاج
npm run build

# معاينة البناء
npm run preview
```

## التقنيات

- **React 19** مع **TypeScript 5.6**
- **Vite 6** (أداء عالي، HMR سريع)
- **ESLint 9** مع TypeScript و React Hooks
- إعداد بناء محسّن: code splitting، minify، target ES2022
